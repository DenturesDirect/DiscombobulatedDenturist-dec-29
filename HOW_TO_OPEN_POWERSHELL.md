## What is PowerShell?

PowerShell is Windows' command-line tool - it's like a text-based way to control your computer. Think of it like a "terminal" or "command prompt."

---

## How to Open PowerShell

### Method 1: Search (Easiest!)

1. **Press the Windows key** (or click Start menu)
2. **Type:** `powershell`
3. **Click on "Windows PowerShell"** or **"PowerShell"**
4. A black or blue window will open - that's PowerShell!

### Method 2: Right-Click Menu

1. **Right-click the Start button** (Windows icon in bottom left)
2. **Click "Windows PowerShell"** or **"Terminal"**

### Method 3: Run Dialog

1. **Press:** `Windows Key + R`
2. **Type:** `powershell`
3. **Press Enter**

### Method 4: File Explorer

1. **Open File Explorer**
2. **Navigate to:** `C:\Users\info\OneDrive\Desktop\Dental_Saas\DentureFlowPro`
3. **Click in the address bar** (where it shows the folder path)
4. **Type:** `powershell`
5. **Press Enter**

This opens PowerShell **already in your project folder!** (This is the best method!)

---

## What PowerShell Looks Like

When you open it, you'll see:
- A window with a dark background (usually black or blue)
- Text that looks like: `PS C:\Users\info>`
- A blinking cursor where you can type

---

## Quick Test

Once PowerShell is open, try typing:
```powershell
cd DentureFlowPro
```

Then press Enter. If it works, you're ready to go!

## Alternative: Use VS Code Terminal

If you're using VS Code (the code editor):
1. **Open VS Code**
2. **Press:** `Ctrl + ~` (Ctrl + tilde key)
3. This opens a terminal at the bottom
4. It's already PowerShell!
